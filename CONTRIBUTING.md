# Kontribusi

Terima kasih telah tertarik untuk berkontribusi pada library IRSMarket API Client!

## Cara Berkontribusi

1. Fork repository ini
2. Buat branch feature (`git checkout -b feature/AmazingFeature`)
3. Commit changes Anda (`git commit -m 'Add some AmazingFeature'`)
4. Push ke branch (`git push origin feature/AmazingFeature`)
5. Buat Pull Request

## Code Style

Library ini menggunakan PSR-12 code style. Jalankan:

```bash
composer phpcs
```

## Testing

Asikkan semua test lolos:

```bash
composer test
```

## Reporting Issues

Jika menemukan bug, silakan buat issue dengan:

- Deskripsi clear tentang bug-nya
- Steps untuk reproduce
- Expected behavior
- Actual behavior
- PHP dan environment version

## Security

Jika menemukan vulnerability keamanan, jangan buat public issue. 
Silakan email ke api-support@aviana.co.id dengan detail.
